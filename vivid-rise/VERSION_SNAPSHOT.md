## 이 버전에서 적용된 내용

### 홈 (Home)
- **남들은 뭘 선택했을까?** 메인 타이틀
- 세 개 카드: **오늘 뭐 먹지** 🍛 / **오늘 뭐 하지** 🏖️ / **나만의 리스트** 📋
- 나만의 리스트 클릭 시 `/#/saved` 이동 (로컬 저장 목록, 백엔드 없음)
- 찜한 장소가 있으면 **❤️ 찜한 장소** 섹션 표시 (결과에서 찜한 곳이 여기 보임)
- 지금 뜨는 실시간 랭킹🔥

### 결과 (Result)
- 지도 + 추천 장소 핀 (오늘 뭐 먹지: 음식 이모지, 오늘 뭐 하지: 활동 이모지)
- 장소 클릭 시 상세 시트: 이름, ⭐, **찜하기(하트)** 🤍/❤️, 상태, 대표 메뉴, 네이버 플레이스에서 보기
- **찜하기** 버튼: 누르면 찜 추가/해제 → 메인 화면·나만의 리스트에 반영
- 하단: **확인** 버튼 (누르면 메인으로) + **메인으로 가기** 링크 (둘 다 메인 이동)

### 나만의 리스트 (/saved)
- 로컬 저장만 사용 (Supabase 연동 없음)
- 찜한 메뉴/장소 목록, 네이버에서 보기, 삭제

### 기술
- Vite + React, HashRouter
- 네이버 지도 API 또는 Leaflet(OpenStreetMap) 폴백
- 앱인토스 SDK 연동 (내비게이션, Safe Area 등)
- `src/utils/favorites.js` — 로컬/앱인토스 Storage 기반 찜

## 이 버전으로 되돌리고 싶을 때

1. **Git 사용 중이면:**  
   `git checkout <이 커밋 해시>` 또는 이 스냅샷 기준 브랜치/태그로 체크아웃.

2. **Git을 아직 쓰지 않았다면:**  
   프로젝트 루트에서 아래로 한 번 커밋해 두면 복원에 유리합니다.
   ```bash
   cd /Users/ad/Desktop/pickt-daily/vivid-rise
   git init
   git add .
   git commit -m "버전 스냅샷: 찜하기(하트) 추가, 확인→메인, 나만의 리스트(로컬)"
   ```
